FAQ

I can't see my prey count!

For prey, make sure that you select Expanded as your game mode when creating a Clan. Classic mode does not have prey count or herbs. You can check your selected game mode on the homepage.

My cats have conditions in Classic mode!

Since ClanGen v0.12, cats have been able to get conditions in Classic mode.

How do I check my allegiances?

Click the Clan's name in the header (near the Next Moon button).

How do I edit a cat?

On a cat's profile page, next to their name, press the pencil icon. It will take you to a page where you can edit their name, roles, etc.

How do I kill a cat?

On a cat's profile page, next to their name, press the exclamation point icon. You will be able to kill or exile the cat.

What's the number next to a cat's name?

That number is the cat's unique ID value, which is used to refer to the cat in the code. On this site, it is also used for the URL of a cat's profile page.

What does custom CSS mean?

It's a way for you to customize the look of the site. For example, you can use custom CSS to change the colours and fonts.

Can I use these cats as OCs?

Yes. However, please do not use the sprites for commercial purposes, as they are licensed under CC BY-NC 4.0.

I lost my Clan! Can you bring it back?

Unfortunately, this is not possible. Clans on this site are are stored in the browser and not backed up on a server.

Can you add [insert mod name here]?

No. Maintaining just the base game is difficult enough.

Can you add more camps?

For practical reasons, currently only camps that are in the ClanGen version used on the site can be used. Please wait for the ClanGen version to be updated.

I found a typo in a patrol!

For practical reasons, typos in ClanGen resource content will not be fixed. Please wait for the ClanGen version to be updated.

When will the website update?

There is no update schedule, but you can see the changelog for a list of past changes. However, the ClanGen version will remain at least one major version behind vanilla ClanGen releases to prevent stealing users from the base game.

How does this site work?

It loads a repackaged version of ClanGen with all its UI removed that can run in the browser, then reads the data from it in order to display it as an ordinary website. The cat sprite drawing code was reimplemented from scratch.
